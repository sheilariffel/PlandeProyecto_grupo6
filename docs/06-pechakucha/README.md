## Reglas del formato PechaKucha

| Regla | Valor |
|-------|-------|
| Cantidad de slides | **20 exactas** |
| Tiempo por slide | **20 segundos automáticos** |
| Tiempo total | **6 minutos 40 segundos** |
| Transición | **Automática** — el presentador se adapta |
| Contenido | **Visual** — imágenes y muy poco texto |

> ⚠️ **Recordatorio:** esto NO es una presentación académica. El objetivo es **convencer** a un sponsor o cliente de que el proyecto vale la pena y que el equipo puede ejecutarlo.

## Criterios de evaluación

| Criterio | Peso |
|----------|:----:|
| Habilidades oratorias (presencia, confianza, contacto visual) | 25% |
| Comunicación persuasiva y capacidad de convencimiento | 25% |
| Coherencia y solidez del plan presentado | 20% |
| Calidad visual y narrativa de las diapositivas | 15% |
| Cumplimiento del formato PechaKucha | 15% |
